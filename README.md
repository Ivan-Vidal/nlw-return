## Como ver o projeto funcionar na sua máquina

### Clone o repositório
`git clone https://github.com/Ivan-Vidal/nlw-return.git`

### Entre no diretorio
`cd web`

### Download dependências
`npm install`

### Rode o Servidor
`npm run dev`

### Está Pronto, Abra seu navegador favorito e navegue até `https://localhost:3000/`
